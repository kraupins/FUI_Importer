# FUI Importer for Unity 6.5

Импортирует `.fui` из Figma-плагина в Unity UI Toolkit.

Структура импорта:

```text
Assets/<PROJECT_NAME>/
  UXML/
  USS/
  Textures/
  Fonts/
  PanelSettings/
  Scenes/
  Info/
```

В окне импортера есть список импортированных проектов и кнопка полного удаления проекта.

Минимальная версия Unity: 6000.5.
