# Changelog

## 1.0.0

- UPM-ready package manifest at repository root.
- Imports `.fui` into UI Toolkit UXML/USS.
- Copies textures and fonts.
- Creates `PanelSettings` asset.
- Creates ready `UIDocument` screen prefabs.
- Optionally creates screen objects in the open scene.
- Supports FUI schema with `Label`, `Button`, `Panel`, `Popup`, `ProgressBar`, `ScrollView`, `Slider`, `CurrencyPanel`, and other visual elements.
